# 隱私權政策

歡迎使用 TAP 售票系統（以下簡稱「本系統」）。我們非常重視您的隱私權，並致力於保護您的個人資料。本隱私權政策說明我們如何收集、使用、披露和保護您的個人資料。

### 1. 個人資料的收集
當您註冊成為會員、購買票券、瀏覽本系統或參與相關活動時，我們可能會收集您的個人資料，包括但不限於：
- 姓名、電子郵件地址、電話號碼、聯絡地址。
- 付款資訊（例如信用卡資訊）。
- 瀏覽紀錄、設備資訊及 IP 位址。

### 2. 個人資料的使用目的
我們收集個人資料的目的在於：
- 處理您的購票訂單、開立電子票券及發送通知。
- 提供及改進本系統的各項會員服務。
- 進行客戶支援、解答您的查詢及處理交易爭議。
- 在取得您同意的前提下，提供行銷、推廣及活動優惠資訊。

### 3. 資料的安全保護
我們採用先進的安全加密防護技術與管理措施，防止您的資料遭受未經授權的存取、篡改、披露或毀損。所有敏感的付款交易均透過安全的伺服器及金流廠商進行加密傳輸。

### 4. 與第三方共用資料
除法律要求或為了完成交易服務（例如將您的必要購票名單提供給該活動之主辦單位）外，我們不會將您的個人資料出售、交易或轉讓給任何第三方。

### 5. 您的權利
您有權查詢、閱覽、要求複製、補充、更正、停止收集、處理、利用或要求刪除您的個人資料。如需行使上述權利，請透過客服信箱（hi@ticket.com）與我們聯絡。
